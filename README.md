# LinguaBiz — Design Kit

Kit de référence pour construire l'app **LinguaBiz** (messagerie multilingue pour entrepreneurs, WhatsApp Business + traduction temps réel) en **React Native (Expo)** sur [blink.new](https://blink.new).

- **`DESIGN.md`** — design system complet (couleurs, typo, composants, 11 écrans)
- **`PROMPT.md`** — prompts prêts à coller dans Blink, dans l'ordre
- **`screenshots/`** — les 11 écrans validés (version verte), à joindre aux prompts
- **`assets/`** — images générées (Nanobanana Pro), optimisées, à utiliser telles quelles dans l'app
- **`reference-html/`** — maquettes HTML pixel-perfect (pour inspecter un détail de style)

Design : fond menthe `#F6FAF7` · vert profond `#0F3D2B` · accent `#1FC16B` · Fraunces + Inter.
